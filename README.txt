# Data Analytics and Machine Learning Portfolio

## Overview
This repository contains a collection of my data analytics, SQL, web scraping, and machine learning projects. The projects demonstrate my ability to clean data, explore datasets, build models, extract insights, and present results in a clear and practical way.

## Projects

### 1. Exploratory Data Analysis for Automotive Service Business Use Case
This project focuses on exploring automotive service data to understand customer behaviour, service patterns, and business-related trends. The analysis helped prepare the dataset for future machine learning model development.

**Skills used:** Python, Pandas, Matplotlib, Seaborn, EDA, data cleaning

---

### 2. Machine Learning Models Development for Automotive Service Business
This project builds machine learning models to predict whether a customer is likely to return for their next automotive service. The project includes data preparation, feature engineering, model training, class balancing, threshold optimization, and feature importance analysis.

**Skills used:** Python, Scikit-learn, SMOTE, Random Forest, model evaluation, Streamlit

---

### 3. Exploratory Data Analysis of Criteria Air Contaminants Using NPRI Canada
This project explores Canada’s National Pollutant Release Inventory dataset to analyze trends in criteria air contaminants across facilities, provinces, cities, and industries.

**Skills used:** Python, Pandas, EDA, data visualization, environmental data analysis, GeoPandas

---

### 4. Machine Learning Models for 5-Year Forecast of CAC Using NPRI Dataset
This project uses machine learning models to forecast criteria air contaminant releases over a five-year period. The model was developed at the facility-year level and predictions were later summarized for interpretation.

**Skills used:** Python, Scikit-learn, regression models, feature engineering, forecasting

---

### 5. SQL Exploratory Data Analysis of COVID Dataset
This project uses SQL to explore COVID-related data and identify patterns in cases, deaths, infection rates, and other important public health indicators.

**Skills used:** SQL, joins, aggregations, filtering, exploratory analysis

---

### 6. Data Scraping from a Random Wikipedia Page
This project demonstrates how web scraping can be used to extract table data from a webpage and convert it into a structured format for analysis.

**Skills used:** Python, BeautifulSoup, Pandas, web scraping

---

### 7. SQL Data Cleaning Using Nashville Housing Dataset
This project focuses on cleaning housing data using SQL. The work includes standardizing dates, handling missing values, splitting address fields, removing duplicates, and preparing the dataset for analysis.

**Skills used:** SQL, data cleaning, string manipulation, duplicate removal

---

### 8. Amazon Web Scraper Project
This project demonstrates how Python can be used to scrape product information from an e-commerce webpage and organize the extracted data into a structured dataset.

**Skills used:** Python, BeautifulSoup, Requests, Pandas, web scraping
